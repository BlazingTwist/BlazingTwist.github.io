A tool for gathering differences between versions of 'School of Dragons'
Developed for the 'School of Dragons' Wiki Team.
The application loads Server-data, then parses and compares it with previous archives.
Comes with several prebuilt subprograms.
Each subprogram defines where data is loaded from, the format of the data, how to access elements that should be compared, and basic post-processing along with where the comparison result should be stored.
Subprograms can be configured from inside the GUI with an automatically generated nested structure.
Don't like the default styling? How about Green or Magenta?
Subprogram results can easily be pasted into spreadsheets.
You can also create subprograms yourself.
